Portal Akses

Pilih Jenis Akses

Tersedia dua jalur akses: kandidat melalui scan barcode, dan operator/admin melalui login terlindungi.

🪪
Portal Kandidat

Diakses melalui scan QR/Barcode. Kandidat mengisi data identitas diri secara mandiri.

  • Input data identitas (NIK, Nama, TTL)
  • Pilih posisi & batch pendaftaran
  • Cepat, mudah, tanpa login
🔐
Portal Operator/Admin

Khusus operator kesehatan & admin. Input data pemeriksaan, MCU, dan pengelolaan data lengkap.

  • Input Kesehatan Dasar (BMI, Tensi, dll)
  • Review & Input Data MCU
  • Manajemen TabelMCU & Tambahan
  • Akses Dashboard Analitik
